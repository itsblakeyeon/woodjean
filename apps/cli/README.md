# woodjean

판교 우드진 카페 단체주문 CLI. 회의용 음료 5~30잔 예약 배달.

## 사용

```bash
npx woodjean order
```

흐름:

1. 도착 시간 선택 (1시간 전 예약, 1시간 단위 슬롯)
2. 메뉴 선택 (시그니처 / 커피 / 논커피 27종)
3. 사이즈 · 온도 · 옵션 · 잔수
4. 배달지 + 닉네임 + 휴대폰
5. 약관 동의 → 접수 완료

결제는 사람과 사람으로 — 배달 시점에 사장님이 현장에서 카드/송금/이체로 받습니다.
