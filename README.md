2. **고르기** — 27종 음료 자유 조합, 1시간 단위 도착 슬롯 (시간당 1건).
3. **확정** — 사장님께 즉시 전달·자동 수락, SMS로 컨펌.
4. **후불** — 도착 시간에 맞춰 배달, 사장님께 카드·송금·이체로 결제.
